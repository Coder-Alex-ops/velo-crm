# Velo CRM

CRM система за велосипеден сервиз — клиенти, велосипеди, ремонти и плащания.

Изградена с Next.js 14 (App Router), TypeScript и Tailwind CSS. Данните се
пазят в локален JSON файл (`data/db.json`) — не е нужна външна база.

## Стартиране

```bash
npm install
npm run dev
```

Отвори [http://localhost:3000](http://localhost:3000).

## Структура

- `src/app` — App Router страници и server actions
- `src/components` — споделени UI компоненти и форми
- `src/lib/db.ts` — JSON persistence слой
- `src/lib/types.ts` — споделени TypeScript типове
- `src/lib/crm.ts` — бизнес помощници (валута, статуси, суми)
- `data/db.json` — съхранение на данните (auto-seed при първи прочит)

## Раздели

- **Dashboard** (`/`) — обобщение, статистики, последни сервизи
- **Клиенти** (`/customers`) — име, телефон, имейл, адрес, бележки
- **Велосипеди** (`/bicycles`) — марка, модел, година, тип, сериен №, рамка
- **Сервизни записи** (`/services`) — описание на работата, части, цени, плащане

## Валута

Всички суми се показват в евро (€).
